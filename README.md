# Car Maintenance & Management System

A robust command-line application built in **C++** designed to manage car services, track parts inventory, and provide detailed cost estimations for automotive businesses.

## 🚀 Key Features

- **Multi-Level Authentication:** Distinct access portals for **Administrators** and **Customers** using custom data structures.
- **Maintenance Scheduling:** Comprehensive management of car services, tracking specific parts and service frequency.
- **Inventory & Parts Tracking:** Real-time viewing of available car models and associated mechanical parts.
- **Detailed Cost Estimation Engine:** An automated calculation system that computes total costs based on parts used, service types, and service counts.
- **Data Persistence:** Utilizes C++ File Streams (`fstream`) to ensure user credentials and car data are saved and loaded correctly from external files.

## 🛠️ Technical Stack

- **Language:** C++17
- **Concepts:** Structured Data (`struct`), File Handling (I/O), Modular Programming, and Control Flow logic.
- **Tools:** Developed and compiled using Visual Studio 2022.

## 📂 Project Structure

- `Source.cpp`: Core logic including user menus, cost calculation algorithms, and authentication functions.
- `Car Renting System.sln`: Visual Studio solution file for environment configuration.
